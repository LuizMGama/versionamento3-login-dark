# Tela de Login Dark

Projeto desenvolvido para a disciplina de Engenharia de Software 3, com foco em versionamento utilizando Git e GitHub.

## Funcionalidades
- Tela de login com tema escuro
- Validação de campos com JavaScript
- Página de dashboard
- Página de cadastro de usuário
- Correção estrutural do HTML principal

## Arquivos do projeto
- `index.html`
- `style.css`
- `script.js`
- `dashboard.html`
- `cadastro.html`

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Git
- GitHub
